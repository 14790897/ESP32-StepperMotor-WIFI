ESP32 ULN2003 步进电机接线图
=====================================

ESP32-C3                    ULN2003驱动板                28BYJ-48步进电机
┌─────────────┐             ┌─────────────┐              ┌─────────────┐
│             │             │             │              │             │
│        GPIO1├─────────────┤IN1          │              │             │
│       GPIO12├─────────────┤IN2          │              │             │
│       GPIO18├─────────────┤IN3          │              │             │
│       GPIO19├─────────────┤IN4          │              │             │
│             │             │             │              │             │
│         GND├─────────────┤GND          │              │             │
│         5V*├─────────────┤VCC          │              │             │
│             │             │             │              │             │
└─────────────┘             │        OUT1├──────────────┤蓝线         │
                            │        OUT2├──────────────┤粉线         │
                            │        OUT3├──────────────┤黄线         │
                            │        OUT4├──────────────┤橙线         │
                            │         VCC├──────────────┤红线         │
                            │             │              │             │
                            └─────────────┘              └─────────────┘

*注意：ESP32的5V引脚可能功率不足，建议使用外部5V电源

外部电源连接方式：
┌─────────────┐
│  5V电源适配器 │
│             │
│        VCC├──────┬─────────ULN2003 VCC
│        GND├──────┼─────────ULN2003 GND
│             │    └─────────ESP32 GND
└─────────────┘

引脚对应表：
===========
ESP32 GPIO  │ ULN2003  │ 功能
───────────┼─────────┼──────────
GPIO1      │ IN1      │ 步进控制1
GPIO12     │ IN2      │ 步进控制2
GPIO18     │ IN3      │ 步进控制3
GPIO19     │ IN4      │ 步进控制4
GND        │ GND      │ 公共地线
5V         │ VCC      │ 电源正极

28BYJ-48电机线色对应：
=====================
线色    │ ULN2003输出 │ 功能
───────┼───────────┼──────
红线    │ VCC        │ 电源正极
蓝线    │ OUT1       │ 线圈A
粉线    │ OUT2       │ 线圈B  
黄线    │ OUT3       │ 线圈C
橙线    │ OUT4       │ 线圈D

步进序列说明：
=============
步骤 │ IN1 │ IN2 │ IN3 │ IN4 │ 说明
────┼────┼────┼────┼────┼──────────
1   │ 1   │ 0   │ 0   │ 0   │ 激活线圈A
2   │ 1   │ 1   │ 0   │ 0   │ 激活线圈A+B
3   │ 0   │ 1   │ 0   │ 0   │ 激活线圈B
4   │ 0   │ 1   │ 1   │ 0   │ 激活线圈B+C
5   │ 0   │ 0   │ 1   │ 0   │ 激活线圈C
6   │ 0   │ 0   │ 1   │ 1   │ 激活线圈C+D
7   │ 0   │ 0   │ 0   │ 1   │ 激活线圈D
8   │ 1   │ 0   │ 0   │ 1   │ 激活线圈D+A

这是8步半步模式，提供更平滑的运动。
正转：按步骤1→2→3→4→5→6→7→8循环
反转：按步骤8→7→6→5→4→3→2→1循环
